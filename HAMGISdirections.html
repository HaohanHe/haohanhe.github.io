系统概述

HAMGIS 是一套完整的地理测量系统，由运行在 Zepp OS 手表上的 HAMGIS 应用和运行在 Android 手机上的 HAMGIS-drop 应用组成。该系统为用户提供高精度的土地测量、数据采集和导出功能，适用于农业、林业、测绘等多个领域。

高精度定位

融合北斗、GPS 多星定位系统，提供厘米级定位精度

自动面积计算

支持多点勾勒测量区域，实时计算面积与周长

高程数据采集

通过气压计采集海拔数据，提供三维地理信息参考

多种导出格式

支持 CSV、JSON、GeoJSON、KML 等多种专业格式

HTTP数据传输

稳定可靠的HTTP数据传输方式

国际化支持

支持中文、英文、日文三种语言

安装指南

HAMGIS（手表端）

  1. 确保您的手表支持 Zepp OS 系统
  2. 安装 Zepp CLI
  3. 克隆 HAMGIS 仓库
  4. 运行 npm install 安装依赖
  5. 使用 npm run build 编译项目
  6. 使用 Zepp CLI 将应用安装到手表

HAMGIS-drop（手机端）

  1. 确保您的设备运行 Android 8.0（API 级别 26）或更高版本
  2. 克隆或下载 HAMGIS-drop 项目
  3. 使用 Android Studio 打开项目
  4. 连接您的 Android 设备或启动模拟器
  5. 点击 "运行" 按钮安装应用

使用说明

手表端操作

工作模式

  • 面积测量模式：专注于农田测量，自动计算面积和周长
  • GIS 采集模式：专业版本，可同时采集点、线、面要素

基本操作

  • 短按 "Collect"：采集点
  • 短按 "Finish"：完成当前要素
  • 长按 "Finish"：完成整个 GIS 项目

手机端操作

数据接收

HTTP 方式
  1. 打开 HAMGIS-drop 应用
  2. 点击 "Start Server" 按钮启动 HTTP 服务器
  3. 确保手表和手机连接到同一局域网
  4. 在手表端选择 "导出到 HTTP" 选项
  5. 手表会自动搜索并连接到手机的 HTTP 服务器
  6. 数据传输完成后，应用会显示 "数据已接收！"

数据导出

  1. 数据接收完成后，应用会显示数据预览
  2. 选择您需要的导出格式（CSV、JSON、GeoJSON、KML）
  3. 选择文件保存位置
  4. 点击 "保存" 按钮完成导出

功能详解

高精度定位

HAMGIS 融合北斗、GPS 多星定位系统，实时追踪测量轨迹，提供厘米级定位精度。在开阔环境下，定位精度可达到 1-3 米，满足大多数土地测量需求。

自动面积计算

系统支持多点勾勒测量区域，自动计算面积和周长。面积计算支持平方米和亩两种单位，方便用户在不同场景下使用。

高程数据采集

通过手表内置的气压计，系统可以采集海拔数据，为用户提供三维地理信息参考。这对于需要考虑地形起伏的测量任务非常有用。

可视化小地图

手表端内置实时地图视图，直观展示测量路径与围合区域。用户可以在测量过程中实时查看测量进度和区域形状。

项目管理

系统支持自动定时测量与完善的历史记录项目管理。用户可以方便地查看、编辑和管理历史测量项目。

导出格式说明

格式 用途 特点 兼容性
CSV 数据表格 包含项目摘要和点位详情，适合数据统计和分析 Excel、Google Sheets、各种数据分析软件
JSON 数据交换 完整的项目数据，结构化存储，易于机器处理 各种编程语言、Web 应用
GeoJSON GIS 分析 标准 GIS 格式，包含空间信息和属性数据 ArcGIS、QGIS、Google Earth Pro
KML 可视化展示 适合在地图上直观展示地理数据 Google Earth、Google Maps、ArcGIS

故障排除

问题：数据未接收

检查 Zepp App 是否已在后台运行，确保手表和手机连接正常。如果使用 HTTP 方式，确保两者在同一局域网内。

问题：导出失败

确保数据格式正确，尝试重新发送数据。如果问题持续存在，检查应用权限设置，确保应用有写入存储的权限。

问题：GPS 信号弱

移至开阔地带，确保手表能接收到足够的卫星信号。首次使用或长时间未使用时，可能需要几分钟时间进行定位。

问题：服务器无法启动

确保应用有相应的网络权限，检查是否有其他应用占用了端口 8888，尝试重启应用。
